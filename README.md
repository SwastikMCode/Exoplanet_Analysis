# Exoplanet Data Analysis

## Overview
This project analyzes exoplanet data from the NASA Exoplanet Archive to study how discovery methods affect observed planet properties.

## Research Question
How do exoplanet orbital period and size differ depending on discovery method?

## Methods
- Python (pandas, matplotlib)
- Data cleaning and filtering
- Log-scale visualization
- Comparative analysis by discovery method

## Key Findings
- Transit method detects mostly short-period planets
- Radial velocity detects a wider range of orbital distances
- Detection method introduces observational bias

## Tools Used
- Python
- Pandas
- Matplotlib

## Dataset
NASA Exoplanet Archive
